# # Amazon Sentiment Analysis

## 📌 Project Overview
This project analyzes Amazon product reviews and classifies them into positive, negative, and neutral sentiments using NLP techniques.

## 🛠 Tools Used
- Python
- Pandas
- TextBlob
- Matplotlib

## 📊 Dataset
Amazon reviews dataset (ratings + review text)

## ⚙️ Process
- Data cleaning
- Sentiment labeling
- NLP analysis
- Visualization

## 📈 Results
- Majority reviews are positive
- Negative reviews highlight delivery and quality issues

## 💡 Conclusion
Sentiment analysis helps businesses understand customer feedback and improve products.
